How to open the form.

There are 3 buttons in the footer, at the bottom of the page: Create, Delete, and Update. Clicking one of these actions will open the form and auto-scroll to it.

This help modal may be obstructing the form and or the actions buttons.

How to use the form.

The form has 4 main sections (boxes). It is designed to make sense when completed in order.

  1. 1. Action. Choose an action: Create, Delete, or Update.
  2. 2. Element. Choose an element: Group or Bookmark.
  3. 3. Detail. Provide details pertaining to the desired action.
  4. 4. Submit. Click the corresponding button to submit the request.

A successfull request will reset the form and reload the bookmarks section. This page does not need to be manually reloaded or refreshed.

Errors are displayed in red, underneath the submit button.

Close the form with the up arrow, located at the bottom edge of the form window. An open form will persist (in local storage) after page refreshes until it is manually closed.

New (empty) database.

The first step is to create a group. A bookmark and it's repsective group share the same database object. This requires a bookmark to be created at the same time as a new group.

Select form fields Create (Action) and Group (Element). The form should already be open with these options preselected.

Add a Group, a Name, and a Url (Detail). Then click Create (Submit).

Rinse and repeat.
